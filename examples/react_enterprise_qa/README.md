# Controlled ReAct Enterprise QA

This is the deterministic, no-key Controlled ReAct Enterprise QA package. It reuses the Enterprise QA knowledge, policy, and tool definitions while adding ReAct planner, review subagent, and response controls for the upcoming governed ReAct workflow.

The manifest validates today, but execution remains fail-closed until the ReAct runtime graph is implemented. Published runs for `react_enterprise_qa` should return `PA_CONFIG_002` with a "not executable yet" message until that runtime task lands.
