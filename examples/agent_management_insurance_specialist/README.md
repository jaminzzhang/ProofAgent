# Agent Management Insurance Specialist Agent

This package configures a staff-facing insurance specialist Agent for internal
office employees who manage agents. It extends the existing Institution
Insurance Specialist pattern with Business Flow Skill Packs for agent basic law,
product clauses, underwriting, claims, customer or agent wording support, and
authorized read-only managed-agent performance/activity lookups.

Business-facing Prompt content is Chinese. Stable Agent Contract keys,
Workflow Stage ids, Tool Contract names, Knowledge Binding ids, Policy Rule ids,
and Business Flow Skill Pack ids remain English.
