# Issue Tracker: GitHub

This repository uses GitHub Issues to track work.

## Skills configuration

- **Provider**: GitHub
- **Tooling**: Uses the `gh` CLI for all operations.

## Workflow

- **Reading issues**: Skills will list and read issues directly from GitHub.
- **Creating issues**: Skills like `to-issues` and `qa` will create new GitHub issues.
- **Updating issues**: Skills will add comments or change labels using `gh issue edit`.
