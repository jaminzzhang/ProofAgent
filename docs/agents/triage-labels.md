# Triage Labels

The `triage` skill uses these labels to manage the lifecycle of issues.

| Role | GitHub Label | Description |
| :--- | :--- | :--- |
| `needs-triage` | `needs-triage` | New issues requiring maintainer evaluation. |
| `needs-info` | `needs-info` | Waiting for more information from the reporter. |
| `ready-for-agent` | `ready-for-agent` | Task is well-defined and can be picked up by an AI agent. |
| `ready-for-human` | `ready-for-human` | Task requires human intervention or complex reasoning. |
| `wontfix` | `wontfix` | Issues that will not be addressed. |
