# Initial Production Client Is Browser Operator Only

Accepted.

[FRAME | HIGH] The initial formal release supports authenticated internal operators through the browser-based Dashboard and Operator Chat only. The CLI remains a local development and controlled maintenance interface, while scripts, CLI clients, and third-party applications receive no direct production API credential or compatibility commitment. Adding a programmatic production client requires a separate support-scope decision, machine-identity model, API lifecycle contract, and release gates.
