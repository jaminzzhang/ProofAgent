# Initial Release Is an Internal Operator Single-Tenant Private Pilot

Accepted.

[FRAME | HIGH] Proof Agent's initial formal release is limited to authenticated internal operators within one organization and one tenant on a private network, using the V3-only Controlled ReAct product baseline. It does not include direct customer self-service, public-internet exposure, or hosted multi-tenant operation; private-network placement does not replace real operator identity or resource authorization. Expanding any excluded boundary requires a new release-scope decision and the corresponding security, data, reliability, and evaluation gates.
